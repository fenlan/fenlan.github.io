еҚіиҰҒеҜ№жҹҘиҜўиҜӯеҸҘ

дёӯзҡ„SELECTеӯ—ж®өиҝӣиЎҢзҢңжөӢ

е°қиҜ•еҲ°3ж—¶еҮәзҺ°No FoundиҜҙжҳҺSELECTйҮҢжңүдёӨдёӘеӯ—ж®өж•°

жҹҘиҜўеҲҡеҲҡзҢңжөӢзҡ„еӯ—ж®өйЎәеәҸ

sqlиҜӯжі•unionжҳҜе°ҶдёӨдёӘйӣҶеҗҲеҺ»йҮҚеҗҲе№¶пјҢзӣёеҪ“дәҺеҺҹжң¬иҝ”еӣһadmin,adminпјҢзҺ°еңЁеҸҲеӯҳеңЁ1,2пјҢйҖҡиҝҮunionеҗҺе°ұжҳҜдёӨиЎҢж•°жҚ®

иҺ·еҸ–еҪ“еүҚж•°жҚ®еә“еҗҚ

database()жҳҜж•°жҚ®еә“зі»з»ҹдҝЎжҒҜеҮҪж•°пјҢдҪңз”ЁжҳҜиҺ·еҸ–еҪ“еүҚж•°жҚ®еә“еҗҚпјҢе…¶д»–ж•°жҚ®еә“дҝЎжҒҜеҮҪж•°жңү

еҮҪж•° дҪңз”Ё
version() иҺ·еҸ–ж•°жҚ®еә“зҡ„зүҲжң¬еҸ·
connection_id() иҺ·еҸ–ж•°жҚ®еә“зҡ„иҝһжҺҘж•°
database();schema() иҺ·еҸ–еҪ“еүҚж•°жҚ®еә“еҗҚ
user();system_user() иҺ·еҸ–еҪ“еүҚз”ЁжҲ·
current_user() иҺ·еҸ–еҪ“еүҚз”ЁжҲ·
charset(str) иҺ·еҸ–еӯ—з¬ҰдёІstrзҡ„еӯ—з¬ҰйӣҶ
collation(str) иҺ·еҸ–еӯ—з¬ҰдёІstrзҡ„еӯ—з¬ҰжҺ’еҲ—ж–№ејҸ

иҺ·еҸ–еҪ“еүҚиЎЁеҗҚ

mysqlеңЁеҲӣе»әж—¶й»ҳи®ӨеҲӣе»әдёҖдёӘinformation_schemaж•°жҚ®еә“пјҢеңЁMySQLдёӯпјҢжҠҠ information_schema зңӢдҪңжҳҜдёҖдёӘж•°жҚ®еә“пјҢзЎ®еҲҮиҜҙжҳҜдҝЎжҒҜж•°жҚ®еә“гҖӮе…¶дёӯдҝқеӯҳзқҖе…ідәҺMySQLжңҚеҠЎеҷЁжүҖз»ҙжҠӨзҡ„жүҖжңүе…¶д»–ж•°жҚ®еә“зҡ„дҝЎжҒҜгҖӮеҰӮж•°жҚ®еә“еҗҚпјҢж•°жҚ®еә“зҡ„иЎЁпјҢиЎЁж Ҹзҡ„ж•°жҚ®зұ»еһӢдёҺи®ҝй—®жқғ йҷҗзӯүгҖӮеңЁINFORMATION_SCHEMAдёӯпјҢжңүж•°дёӘеҸӘиҜ»иЎЁгҖӮе®ғд»¬е®һйҷ…дёҠжҳҜи§ҶеӣҫпјҢиҖҢдёҚжҳҜеҹәжң¬иЎЁпјҢеӣ жӯӨпјҢдҪ е°Ҷж— жі•зңӢеҲ°дёҺд№Ӣзӣёе…ізҡ„д»»дҪ•ж–Үд»¶гҖӮ

information_schemaж•°жҚ®еә“иЎЁиҜҙжҳҺ:

  • SCHEMATAиЎЁпјҡжҸҗдҫӣдәҶеҪ“еүҚmysqlе®һдҫӢдёӯжүҖжңүж•°жҚ®еә“зҡ„дҝЎжҒҜгҖӮshow databasesзҡ„з»“жһңеҸ–д№ӢжӯӨиЎЁгҖӮ
  • TABLESиЎЁпјҡжҸҗдҫӣдәҶе…ідәҺж•°жҚ®еә“дёӯзҡ„иЎЁзҡ„дҝЎжҒҜпјҲеҢ…жӢ¬и§ҶеӣҫпјүгҖӮиҜҰз»ҶиЎЁиҝ°дәҶжҹҗдёӘиЎЁеұһдәҺе“ӘдёӘschemaпјҢиЎЁзұ»еһӢпјҢиЎЁеј•ж“ҺпјҢеҲӣе»әж—¶й—ҙзӯүдҝЎжҒҜгҖӮshow tables from schemanameзҡ„з»“жһңеҸ–д№ӢжӯӨиЎЁгҖӮ
  • COLUMNSиЎЁпјҡжҸҗдҫӣдәҶиЎЁдёӯзҡ„еҲ—дҝЎжҒҜгҖӮиҜҰз»ҶиЎЁиҝ°дәҶжҹҗеј иЎЁзҡ„жүҖжңүеҲ—д»ҘеҸҠжҜҸдёӘеҲ—зҡ„дҝЎжҒҜгҖӮshow columns from schemaname.tablenameзҡ„з»“жһңеҸ–д№ӢжӯӨиЎЁгҖӮ
  • STATISTICSиЎЁпјҡжҸҗдҫӣдәҶе…ідәҺиЎЁзҙўеј•зҡ„дҝЎжҒҜгҖӮshow index from schemaname.tablenameзҡ„з»“жһңеҸ–д№ӢжӯӨиЎЁгҖӮ
  • USER_PRIVILEGESпјҲз”ЁжҲ·жқғйҷҗпјүиЎЁпјҡз»ҷеҮәдәҶе…ідәҺе…ЁзЁӢжқғйҷҗзҡ„дҝЎжҒҜгҖӮиҜҘдҝЎжҒҜжәҗиҮӘmysql.userжҺҲжқғиЎЁгҖӮжҳҜйқһж ҮеҮҶиЎЁгҖӮ
  • VIEWSиЎЁпјҡз»ҷеҮәдәҶе…ідәҺж•°жҚ®еә“дёӯзҡ„и§Ҷеӣҫзҡ„дҝЎжҒҜгҖӮйңҖиҰҒжңүshow viewsжқғйҷҗпјҢеҗҰеҲҷж— жі•жҹҘзңӢи§ҶеӣҫдҝЎжҒҜ
  • TRIGGERSиЎЁпјҡжҸҗдҫӣдәҶе…ідәҺи§ҰеҸ‘зЁӢеәҸзҡ„дҝЎжҒҜгҖӮеҝ…йЎ»жңүsuperжқғйҷҗжүҚиғҪжҹҘзңӢиҜҘиЎЁ

жҳҺзҷҪжіЁе…Ҙзҡ„еҹәжң¬ж–№ејҸе’ҢеҺҹзҗҶпјҢеҫҲе®№жҳ“зҹҘйҒ“пјҢйҖҡиҝҮиҫ“е…ҘжЈҖжҹҘжқҘйў„йҳІжіЁе…ҘгҖӮиҺ·еҸ–еҲ°иҫ“е…Ҙзҡ„еӯ—з¬ҰдёІпјҢеҜ№еӯ—з¬ҰдёІиҝӣиЎҢиҝҮж»ӨгҖҒиҪ¬д№үзӯүж“ҚдҪңпјҢеҸӘдёҚиҝҮиҜёеҰӮphpзҡ„дёҚеҗҢиҫ“е…ҘжЈҖжҹҘеҮҪж•°пјҢдёҚеҗҢзҡ„еҮҪж•°зҡ„жЈҖжҹҘж–№жі•дёҚдёҖж ·пјҢдҪҶжңүеҸҜиғҪд»Қз„¶еӯҳеңЁзј–еҶҷиҖ…жң¬иә«жІЎжңүиҖғиҷ‘еҲ°зҡ„еҸҜиғҪжҖ§пјҢиҝҷе°ұеҜјиҮҙдәҶжјҸжҙһдёҚй—ҙж–ӯеҮәзҺ°гҖӮеңЁдҪҝз”ЁиҝҷдәӣжЈҖжҹҘеҮҪж•°ж—¶пјҢиҰҒи®ӨзңҹжҹҘзңӢеҮҪж•°зҡ„жЈҖжҹҘж–№жі•д»ҘеҸҠиҝҮж»Өзҡ„дёңиҘҝгҖӮеңЁжӯӨдёҫдёӘдҫӢеӯҗ:

mysqli_real_escape_string( mysqli $link , string $escapestr):

  • Parameters:
    • link: Procedural style only: A link identifier returned by mysqli_connect() or mysqli_init()
    • escapestr: The string to be escaped. Characters encoded are NUL (ASCII 0), \n, \r, \, вҖҳ, вҖң, and Control-Z.
  • Return Value:
    • Returns an escaped string.
